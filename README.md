# Advent of Code

These are my solutions to the Advent of code challenges.

## Requirements

- [Deno](https://deno.com/)

## Running

Run with:

```
deno run -A /index.ts
```

You can also pass a year

```
deno run -A /index.ts 2023
```

or a year and a day

```
deno run -A /index.ts 2023 1
```
